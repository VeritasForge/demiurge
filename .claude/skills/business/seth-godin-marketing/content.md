# 콘텐츠 전략 가이드 — 세스 고딘 철학 기반

## 콘텐츠의 목적

콘텐츠는 광고가 아니다. 콘텐츠는 **관대함의 표현**이다. 좋은 콘텐츠는 아무것도 요구하지 않으면서 읽는 사람의 세계를 조금 더 낫게 만든다. 그 관대함이 쌓이면 신뢰가 되고, 신뢰가 쌓이면 퍼미션이 된다.

## 콘텐츠 유형

### 1. 등대 콘텐츠 (Lighthouse Content)

같은 가치관을 가진 사람들을 끌어당기는 콘텐츠. "우리는 이것을 믿습니다"라는 선언.

특징:
- 업계의 통념에 도전한다
- 명확한 관점(POV)이 있다
- 일부는 동의하지 않을 수 있다 (그래야 정상)
- 공유하고 싶은 충동을 일으킨다

예시: "우리는 A/B 테스트를 하지 않습니다. 이유는 이렇습니다."

### 2. 선물 콘텐츠 (Gift Content)

순수하게 도움이 되는 콘텐츠. 아무 조건 없이 제공한다.

특징:
- 실용적이고 바로 적용 가능하다
- 숨김 없이 모든 것을 공개한다
- "이걸 무료로 줘도 돼?" 싶을 정도로 관대하다
- 검색과 공유로 자연스럽게 퍼진다

예시: 업계 노하우 가이드, 무료 템플릿, 체크리스트

### 3. 연결 콘텐츠 (Connection Content)

트라이브 구성원들이 서로를 발견하고 연결되게 하는 콘텐츠.

특징:
- 대화를 시작한다
- 공동체 의식을 강화한다
- 참여를 유도한다 (질문, 챌린지, 공유)
- 커뮤니티의 문화를 형성한다

예시: "당신의 ~경험을 들려주세요", 커뮤니티 하이라이트

## 콘텐츠 제작 프레임워크

### 세스 고딘의 블로그 공식

세스 고딘은 7,000개 이상의 블로그 포스트를 매일 빠짐없이 썼다. 그의 공식:

1. **하나의 아이디어**: 한 편에 하나의 생각만
2. **짧게**: 길어야 300단어 (한국어 기준 600자)
3. **매일**: 일관성이 품질보다 중요하다
4. **의견을 가져라**: "어떤 사람들은 ~라고 합니다"가 아니라 "이것은 ~입니다"
5. **행동을 암시하라**: 명시적 CTA 없이도 "그래서?"가 담겨있게

### 콘텐츠 달력 설계

주간 리듬 예시 (블로그/뉴스레터 기준):

- 월: 등대 콘텐츠 (관점 제시)
- 수: 선물 콘텐츠 (실용적 가이드/팁)
- 금: 연결 콘텐츠 (이야기, 사례, 커뮤니티)

핵심: 빈도보다 **일관성**. 주 1회를 약속했으면 매주 1회를 반드시 지킨다.

## 콘텐츠 배포

### 자체 채널 우선 원칙

1. 블로그/웹사이트 (소유 자산)
2. 이메일 뉴스레터 (직접 도달)
3. 소셜 미디어 (발견의 창구)

소셜 미디어는 **유입 채널**이지 **본거지**가 아니다. 알고리즘이 바뀌어도 이메일 리스트와 블로그는 남는다.

### 입소문 설계

콘텐츠가 공유되려면:
- 공유한 사람이 **똑똑해 보여야** 한다
- 받는 사람에게 **즉각적 가치**가 있어야 한다
- **쉽게 공유**할 수 있어야 한다 (링크 하나로)

"이 콘텐츠를 공유하면 공유한 사람의 지위가 올라가는가?" — 이 질문에 "예"여야 한다.

## 콘텐츠 품질 체크리스트

- [ ] 하나의 명확한 메시지가 있는가?
- [ ] 타겟 독자가 처음 3초 안에 "이건 나를 위한 거다"라고 느끼는가?
- [ ] 읽고 나서 뭔가 하나라도 새롭게 알게 되거나 다르게 생각하게 되는가?
- [ ] 공유했을 때 공유한 사람이 멋져 보이는가?
- [ ] 브랜드를 가려도 우리 목소리라는 것을 알 수 있는가?
- [ ] 과장이나 클릭베이트 요소가 없는가?
