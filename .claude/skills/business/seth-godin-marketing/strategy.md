# 마케팅 전략 가이드 — 세스 고딘 철학 기반

## 전략 수립 프로세스

### Phase 1: 공감과 이해

전략은 스프레드시트가 아니라 **사람에 대한 이해**에서 시작한다.

#### 공감 지도 작성법

```
          [생각하고 느끼는 것]
         내면의 걱정, 열망, 꿈
              |
[듣는 것] ——— 고객 ——— [보는 것]
주변의 조언,        시장 환경,
또래 행동         경쟁사 메시지
              |
          [말하고 행동하는 것]
         실제 행동, 태도, 외면적 모습
```

핵심: "말하고 행동하는 것"과 "생각하고 느끼는 것" 사이의 **간극**이 마케팅 기회다.

#### 지위 역할 분석

세스 고딘에 따르면, 모든 구매 결정에는 지위(Status)가 관여한다:

- **소속(Affiliation)**: "나도 그 그룹의 일원이다" → 커뮤니티, 멤버십, 브랜드 로열티
- **지배(Dominance)**: "나는 남들보다 앞서 있다" → 프리미엄, 얼리어답터, 독점

타겟 고객이 어떤 지위 동기로 움직이는지 파악하라. 이것이 메시지 프레이밍을 결정한다.

### Phase 2: 최소 유효 시장 설계

#### 왜 좁혀야 하는가

넓은 시장을 노리면:
- 메시지가 희석된다
- 경쟁이 치열하다
- 입소문이 퍼지지 않는다 (서로 모르는 사람들이니까)

좁은 시장을 노리면:
- 메시지가 날카롭다
- 그 안에서 1등이 될 수 있다
- 입소문이 자연스럽다 (서로 아는 사람들이니까)

#### 최소 유효 시장 정의 템플릿

```
우리의 최소 유효 시장은:
- [인구통계]: 
- [심리통계]: ~을 믿고, ~을 두려워하며, ~을 열망하는
- [행동]: 현재 ~하고 있으며, ~에 불만을 느끼는
- [규모]: 약 ~명 (구체적 숫자)
- [모이는 곳]: 온라인/오프라인 어디서 만날 수 있는가
```

### Phase 3: 포지셔닝

#### XY 포지셔닝 맵

두 가지 축을 선택하여 경쟁 환경을 시각화한다.

축 선택 기준:
- 고객이 실제로 **신경 쓰는** 기준이어야 한다 (업계가 중요시하는 기준이 아니라)
- 아직 아무도 차지하지 않은 사분면이 있어야 한다
- 그 사분면에서 1등이 될 수 있어야 한다

#### 포지셔닝 문장

```
[타겟]을 위한 [카테고리]로서,
[경쟁 대안]과 달리
우리는 [핵심 차별점]을 제공합니다.
왜냐하면 [믿을 수 있는 이유]가 있기 때문입니다.
```

주의: 차별점이 "더 좋다", "더 싸다"이면 포지셔닝이 아니다. **다르다**여야 한다.

### Phase 4: 채널 전략

#### 퍼미션 기반 채널 우선순위

1순위 — 동의 기반 채널 (소유 미디어):
- 이메일 뉴스레터, 블로그, 팟캐스트, 커뮤니티

2순위 — 공유 기반 채널 (공유 미디어):
- 입소문, 리퍼럴, 사용자 생성 콘텐츠

3순위 — 관계 기반 채널 (획득 미디어):
- PR, 파트너십, 게스트 콘텐츠

4순위 — 유료 채널 (구매 미디어):
- 광고 (단, 퍼미션을 얻기 위한 수단으로만 사용)

원칙: 유료 광고의 목적은 **판매가 아니라 퍼미션 획득**이다. 광고로 바로 구매를 유도하지 말고, 이메일 구독이나 무료 체험 등 관계의 시작점으로 활용한다.

### Phase 5: 측정

#### 세스 고딘식 지표

전통적 지표 대신 관계와 신뢰를 측정하는 지표를 우선한다:

| 허영 지표 (Vanity) | 의미 있는 지표 (Meaningful) |
|---------------------|----------------------------|
| 페이지뷰 | 재방문율 |
| 팔로워 수 | 댓글/공유율 |
| 이메일 리스트 크기 | 오픈율, 답장률 |
| 다운로드 수 | 완료율, 재사용율 |
| 도달(Reach) | 전환된 진성 팬 수 |

핵심 질문: "이 숫자가 올라가면, 실제로 우리 비즈니스가 나아지는가?"

## 전략 문서 출력 형식

마케팅 전략 문서를 요청받으면 다음 구조를 따른다:

```
# [프로젝트명] 마케팅 전략

## 1. 우리는 누구를 위해 존재하는가
   - 타겟 페르소나
   - 공감 지도 요약
   - 지위 동기 분석

## 2. 우리는 어떤 변화를 만드는가
   - Before → After
   - 핵심 약속 (하나의 문장)

## 3. 우리는 어떻게 다른가
   - 포지셔닝 맵
   - 포지셔닝 문장
   - "우리는 ~이 아닙니다" (명확한 경계)

## 4. 우리는 어떻게 연결하는가
   - 채널 전략 (우선순위)
   - 퍼미션 획득 전략
   - 콘텐츠 전략 개요

## 5. 우리는 무엇을 측정하는가
   - 핵심 지표 3개
   - 측정 주기와 방법

## 6. 90일 실행 계획
   - Month 1: 기반 구축
   - Month 2: 실행과 학습
   - Month 3: 최적화와 확장
```
